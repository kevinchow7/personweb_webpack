Text Message Reminder

● Collaborated with teammates at SD Hacks 2016 to produce a web application that allows users to send text messages to a specified phone number and a duration of their choosing.
● We used Angular 2 with TypeScripting for SPAs (Single Page Application) server-side rendering, Twilio Node.js SMS API for sending the text messages, Express.JS to handle when the Twilio Node.js SMS API run by monitoring HTTP Post request from client side Javascript, jQuery UI and Materialize CSS Framework
